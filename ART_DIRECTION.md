# BLACKLINE — Art Direction

## Direction
Premium neo-brutalism / global logistics control interface.

## Core metaphor
A freight route is not a dotted line on a map. It is a living system of handoffs, capacity, risk and recovery decisions.

## Visual language
- industrial black background;
- signal red used as an operational state, not decoration;
- oversized compressed typography;
- hard borders, terminal tables and route coordinates;
- animated global corridors;
- moving container rail;
- deliberate contrast between dark control screens and warm paper sections.

## Key interactions
- cursor-responsive controls;
- route map motion;
- clickable global hubs;
- route strategy selector;
- live shipment timeline;
- scroll-driven container convoy;
- exception control board.
