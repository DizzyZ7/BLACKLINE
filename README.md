# BLACKLINE

Premium interactive landing page for an international logistics and freight-control brand.

## Run locally

```bash
npx serve .
```

Open the local URL shown in the terminal.

You may also open `index.html` directly, but a local HTTP server is recommended. Internet access is required for GSAP and Lenis CDN scripts.

## Structure

- `index.html` — semantic page markup and brand content
- `styles.css` — full visual system and responsive design
- `scripts/app.js` — interactions, route engine, tracking demo and motion
- `assets/favicon.svg` — brand favicon
- `ART_DIRECTION.md` — visual concept
- `CASE_STUDY_RU.md` — Russian case description
- `REELS_SHOT_LIST_RU.md` — suggested recording sequence

## Status

Original concept project. The public website itself contains no portfolio or concept labels.
